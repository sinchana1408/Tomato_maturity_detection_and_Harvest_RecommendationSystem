# 🍅 TomatoCare AI

## 📌 Project Description
TomatoCare is an AI-based system that detects tomato ripeness using multiple CNN models and provides harvest recommendations.

## 🚀 Features
- Multi-model prediction (ResNet50, MobileNetV2, DenseNet121, EfficientNetB0)
- Confidence comparison
- Best model selection
- Harvest recommendation system
- Visualization dashboard

## 🧠 Models Used
- ResNet50
- MobileNetV2
- DenseNet121
- EfficientNetB0
